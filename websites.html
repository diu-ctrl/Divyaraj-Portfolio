Case Study • E-Commerce

Abhushan Jewelry

"A luxury, responsive e-commerce experience and appointment scheduler — complete with an administrative manager — for fine gold and permanent bridal jewelry in Ahmedabad."

Role Full-Stack Design & Dev
Timeline 4 Days
Status Live
Platform Vercel (Edge Network)
Visit Live Site View on GitHub

The Challenge

Abhushan is a fine gold jewelry studio in Ahmedabad. They had no digital presence that matched their in-studio luxury experience. High-value clients were dropping out of the sales funnel due to manual scheduling friction for custom design consultations. They needed a luxury digital showcase combined with a seamless studio booking engine, backed by an admin portal to manage appointments in real time.

My Role & Scope

  • Visual identity system (dark cocoa '#1E1617', gold accent '#D4AF7A', editorial Garamond & Playfair typography)
  • Interactive high-fidelity category grid for gold necklaces, solid gold rings, and custom bridal collections
  • 4-service studio appointment booking scheduler (Welded Forever Bracelet, Custom Engagement, Bridal Consultation, Jewelry Refurbishing)
  • Dynamic time-slot selector and real-time slot validation
  • SEO infrastructure (Schema.org: LocalBusiness, Product, FAQPage)
  • Core Web Vitals optimization (compressed WebP, font preloading)
  • Full admin dashboard: credentials login wall, metrics dashboard (Total, Pending, Approved, Completed, Cancelled), search/filter by customer/service, WhatsApp confirmation triggers, and CSV export
  • Live deployment on Vercel Edge network

Case Study • Automotive

Detailing Cars

"A premium, mobile-first auto detailing booking platform — complete with a live admin dashboard — designed to streamline vehicle care scheduling in Ahmedabad."

Role Lead Design & Frontend Dev
Timeline 3 Days
Status Live
Platform Vercel (Edge Network)
Visit Live Site View on GitHub

The Challenge

Detailing Cars is a premium mobile automotive styling studio in Ahmedabad, offering ceramic coating, paint correction, and full interior detailing. The business was losing potential bookings due to a manual, non-responsive pricing sheet and no digital presence. They needed a complete online system — a high-conversion landing page with instant pricing, plus an admin dashboard to manage and track every booking in real time.

My Role & Scope

  • Full visual identity system (dark mode, gold/warm-cream palette, geometric typography)
  • Interactive paint-restoration before/after slider
  • 3-tier vehicle class dynamic pricing engine (Hatchback / Sedan / SUV)
  • Floating capsule navbar with services dropdown & seasonal offer bar
  • SEO infrastructure (Schema.org: Service, AutomotiveBusiness)
  • Core Web Vitals optimization (compressed WebP, font preloading)
  • Full admin dashboard: metrics grid, bookings table, calendar view, weekly chart, filter & export
  • Password-protected admin login with toast notifications
  • Edge caching deployment on Vercel

Case Study • Refined Fashion

KXM Fashion

"A high-concept, minimalist editorial fashion storefront — featuring a custom dynamic cart drawer and fluid horizontal lookbooks — designed to present a refined clothing line with intention and freedom."

Role Front-End Design & Dev
Timeline 3 Days
Status Live
Platform Vercel (Edge Network)
Visit Live Site View on GitHub

The Challenge

KXM is an editorial, culture-driven clothing brand. Standard grid-based e-commerce layouts failed to reflect their message of intention and quiet luxury. They needed a digital experience that felt like a high-end editorial magazine, combining large typographic statements and seamless infinite horizontal carousels with a fast, responsive slide-out shopping bag and local storage cart persistence.

My Role & Scope

  • Custom font architecture implementing PP Neue Montreal for UI elements and PP Editorial New for large editorial statements
  • Infinite horizontal product marquee track controlled by clean, hardware-accelerated CSS animations
  • Interactive hover transitions on products showing alternate angles and garment drape styles
  • Dynamic slide-out shopping cart drawer showing live item updates and subtotal calculations
  • E-commerce state management using browser localStorage/sessionStorage APIs to persist shopping bag choices
  • Full-bleed lifestyle lookbook modules and responsive layouts optimized across mobile and desktop devices
  • Core Web Vitals tuning: responsive asset loading, font preloading, and high-performance layout rendering
  • Live deployment and configuration on Vercel's global edge network

Key Features

Feature Description Technical Note
Editorial Typography Aesthetic font pairing of PP Neue Montreal and PP Editorial New serif styles @font-face loading, fluid line heights, high-contrast layouts
Infinite Product Marquee Smooth, horizontal scrolling featured products track CSS keyframe transform loop with seamless coordinate wrap
Garment State Hover Alternate product display on mouseover showing details/styling Dual absolute image tags, CSS opacity and scale transitions
Dynamic Cart Drawer Slide-out bag for items list, quantity updates, and pricing updates Vanilla JS DOM events, event listeners, dynamic subtotals
Manifesto Layout Brutalist full-width quotes and vertical content sections CSS grid, clamp text values, responsive spacing
State Persistence E-commerce checkout data persists across page sessions localStorage parsing, stringified cart states
